Overview

오늘의 집? 오늘의 팀!

"안녕하세요. 정창식 입니다."

Backend, Frontend, iOS까지 총 10명이 한 팀으로 구성하여 하나의 서비스를 프론트부터 서버까지 만들어 보는 프로젝트
현재 매우 빠르게 성장중인 스타트업인 버킷플레이스가 서비스하고 있는 인테리어 플랫폼 `오늘의 집`을 클론한 프로젝트
  • 기간 : 2019.07.01 ~ 2019.08.09 (5주)
  • 소속 : 회식First팀 내 iOS개발팀 팀장.
  • 역할
    • 다른 팀과의 의사소통 전반
    • 자체 로그인과 소셜로그인(카카오, 구글, 네이버)등 앱 내에서 권한이 필요한 작업들 전반
    • 커스텀 UI 추상화 시키고 여러 화면에 적용할 수 있게 만들기(CustomCategoryTabBar)
    • Extension 과 Helper Function 적용시키기
    • 마이페이지와 사용자 프로필 전반 담당

CHUNGBUK NATIONAL UNIVERSITY

충북대학교 정보통신공학부 졸업
2011.03 - 2019.02

제 고향인 충북의 거점 국립대학교인 충북대학교에 성적우수장학생으로 입학했습니다. 컴퓨터와 인터넷이 너무 신기해서 IT 관련 학과인 정보통신공학부로 전공을 선택했습니다.
전역 후 활발한 성격으로 단과대학 대의원회 부의장을 맡으며 학생회와 일반 학생들 사이의 커뮤니케이션을 조절하는 역할을 하였습니다. 대학생 학습 멘토링으로 교육 봉사를 하고 장학금을 받은 경험이 있습니다. 이후에는 학술동아리(ES2) 회장을 맡아 매주 세미나를 진행하는 등 대학 생활의 많은 부분을 여러 사람을 만나고 커뮤니케이션 하는 시간으로 보냈습니다.
대학교 3학년 겨울에 프로그래밍을 제대로 접한 후 즉시 모든 커뮤니티 활동을 중단하고 프로그래밍 수업 위주로 수강 신청하고, 프로그래밍 공부에만 집중했습니다.

도서관

36번의 도서대출
2017.11 - 2019.02

도서관이 없었다면 저는 이렇게 단기간에 집중해서 컴퓨터를 배우지 못했기 때문에 도서관은 저에게 하나의 교육을 수료한 것이나 마찬가지입니다. C 언어 수업 하나만 듣고 프로그래밍에 대해 아무것도 모를 때 내가 어디로 가야 하는지 조언해 주는 멘토가 되어주고, 새로운 언어인 Swift를 배우고 싶어 할 때는 그것을 알려주는 선생님이 되어주었기 때문입니다. '프로그래밍이 내 길이다'라고 생각한 이후 도서관에서 1년 동안 프로그래밍 관련 서적을 약 36번 대출하면서 지식을 빠르게 흡수했습니다.

MIT - Introduction to Computer Science and Programming Using Python

MITx: 6.00.1x Introduction to Computer Science and Programming Using Python
2019.01 - 2019.04

edX를 통해 MIT에서 하는 컴퓨터과학 기초수업을 수강할 수 있었습니다. 이 수업을 수강하기 위해 영어 공부를 2개월 동안 집중적으로 투자할 만큼 컴퓨터공학 기초에 많은 투자를 했습니다. 대학교 4학년의 마지막 수업을 마치자마자 바로 시작하고 싶었지만, 미국 학기를 시작하는 시점이 한국과는 다르기 때문에 저는 한 한기 이전인 2018년 9월 강의를 수강했습니다. 이 수업을 통해 컴퓨터의 역사를 시작으로, 파이썬 언어, 기초적인 자료형과 자료구조, 기본적인 알고리즘과 빅-Oh 알고리즘 분석, 객체지향 프로그래밍, 디버깅의 원리와 핵심 등을 학습했습니다. 저는 이 시기에 프로그래밍 실력이 가장 많이 늘었다고 생각합니다.

    패스트캠퍼스 iOS 프로그래밍 스쿨 수강
    2019.04 - 현재

    Swift와 iOS의 이론적인 부분이 자신이 생길 시점에 실제 프로젝트 위주로 공부가 필요하다고 판단하여 대학을 졸업하자마자 패스트 캠퍼스 iOS 프로그래밍 스쿨을 수강하여 애플의 iOS 시스템을 체계적으로 배웠습니다. GiftBot의 CEO 이봉원 강사님에게 코드를 효율적으로 짜는 방법부터 코드 스타일, iOS를 다루는 노하우, 정확한 정보를 얻는 방법 등 혼자 공부하며 배울 수 없었던 많은 것들을 배웠습니다. 해커톤이라는 행사를 통해 2번의 개인 프로젝트를 하면서 완전한 앱을 시작부터 끝까지 만들어 보는 경험을 했고 이때 만든 앱을 앱스토어에 배포하고 있습니다. Frontend, Backend와 협업할 수 있는 팀 프로젝트를 하면서 협업과 커뮤니케이션 중요성을 배웠습니다. 협업을 효율적으로 하고자 slack, Github, trello 등 협업 툴을 적극적으로 사용하여 그 사용법을 잘 알고 있습니다.
